# PokeApp

PokeApp es una aplicación web que permite visualizar una lista de los primeros 151 Pokémones, ver el detalle de cada Pokémon, buscar Pokémones y gestionar una lista de Pokémones "Listo para el combate". Esta aplicación fue desarrollada utilizando React, TypeScript y Redux, y se despliega en Vercel.

## Tecnologías Utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Redux](https://redux.js.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [PokeApi](https://pokeapi.co/)
